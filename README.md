# Louvre

## [Live preview](https://evgeniyadanilovich.github.io/louvre/)

An adaptive landing page with a lot of features.

## Main features
* Slider
* Slider before/after
* Castom videoplayer
* Paralax effect

## Technologies
JavaScript, HTML, SCSS
